# traffic-congestion
Traffic Congestion Prediction and Visualization System

## 프로젝트 개요

전기화물차의 센서 데이터와 GPS 위치 정보를 활용하여 도로의 교통 혼잡도를 예측하고 시각화하는 머신러닝 프로젝트이다.

차량에서 수집되는 주행 데이터와 위치 정보를 기반으로 교통 상태를 "원활", "서행", "정체"로 분류하고, 지도 기반 시각화를 통해 사용자가 직관적으로 교통 상황을 확인할 수 있도록 구현하였다.

---

## 프로젝트 목표

- 차량 센서 데이터를 활용한 교통 혼잡도 분류
- GPS 기반 위치 정보와 공간 특성 반영
- 지역별 혼잡 위험도 산출
- 실시간 혼잡도 예측 시스템 구현
- 지도 기반 시각화 제공

---

## 사용 데이터

### 데이터 출처

- 한국교통안전공단 전기화물차 센서 데이터

### 주요 변수

- GPS 위도(lat)
- GPS 경도(lon)
- 가속도(acceleration)
- 제동 정보(braking)
- 차량 제어 정보
- 모터 속도
- 모터 토크
- 방위각
- 고도

---

## 데이터 전처리

### 위치 데이터 변환

GPS 좌표를 활용하여 위치 정보를 생성하였다.

### 공간 특성 생성

격자(Grid) 기반 공간 분할을 수행하여

- grid_id 생성
- 지역별 혼잡 위험도(grid_congestion_risk) 생성

을 수행하였습니다.

이를 통해 특정 지역이 평소 얼마나 혼잡한지에 대한 공간 정보를 모델에 반영하였다.

### 데이터 스케일링

- StandardScaler 적용

---

## 머신러닝 모델

### Random Forest Classifier

교통 상태를 다음 3개 클래스로 분류하였습니다.

| 클래스 | 설명 |
|----------|----------|
| 원활 | 교통 흐름이 원활한 상태 |
| 서행 | 차량 속도가 감소한 상태 |
| 정체 | 심각한 교통 혼잡 상태 |

---

## 모델 성능

### Classification Report

| Class | Precision | Recall | F1-score |
|---------|---------|---------|---------|
| 정체 | 1.00 | 0.97 | 0.99 |
| 서행 | 0.55 | 0.95 | 0.69 |
| 원활 | 0.93 | 0.92 | 0.92 |

### Overall Performance

- Accuracy : 97%
- Macro F1-score : 0.87
- Weighted F1-score : 0.97

## 시스템 구현

### 실시간 혼잡도 예측

사용자가 입력한 차량 정보와 위치 정보를 기반으로

- 혼잡도 예측
- 혼잡 위험도 평가

를 수행할 수 있도록 구현하였다.

### Streamlit Dashboard

Streamlit을 활용하여 웹 기반 인터페이스를 구축하였다.

### 지도 시각화

Folium을 활용하여

- 위치 정보 시각화
- 지역별 혼잡도 표시

기능을 구현하였다.
